A facet descriptor is a XML file defining the HTML rendering for a stencil.

The facets are defined for the following type : label, panel, tree, debug. The synatx is defined in {@link com.gdo.facet.facets.dtd}.

The rendering is defined locally as a CData field or in a file thru the file attribute. For each type, a default value can be defined when the default attribute is set to true. Otherwise the first description serves as the default one.
