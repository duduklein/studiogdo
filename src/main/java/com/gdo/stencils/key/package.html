Unique slot key generator.

When a stencil is plugged in a {@link com.gdo.stencils.slot.MultiSlot} with a key, this key must be unique. To be sure that this key is unique, if it is a string, this key can be encapsulated in a IKeyGenerator object, which will give another possible values if another stencil is already plugged into the slot at this key.

Two generators are defined in this package :

  • IntKeyGenerator: For an integer key.
  • StringKeyGenerator: For a String key.
